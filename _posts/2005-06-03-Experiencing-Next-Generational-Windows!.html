前段时间的 Microsoft WinHEC 2005 盛会上，Bill Gates 亲自演示了处于紧张开发中的下一代 Windows 操作系统，就是网上炒得沸沸扬扬的 Longhorn 5048。

因为没有方便我使用的 DVD 刻录机，系统的大小又超过了普通的 CD-ROM 光盘。经过一段时间的研究，今天终于成功实现了从硬盘安装。(已贴上界面截图)

 

我从 3683 版本开始试用 Longhorn，一直到后来的 4008、4015、4029、4051、4053、4074!!和现在的5048。

从某一意义来说我也是见证 Longhorn 一路成长的人之一。感觉 Longhorn 虽然到今天还不是很完善，但是就像几年前我们疯狂想要得到 Windows XP 的测试版一样，Longhorn 将延续甚至超越 XP 的辉煌我认为是一个可以预见的铁定事实。就像那个时候，我的机子在 815E/C2-1G/128MSDR/30G-HD 的电脑上勉强运行 RC 版本的 Windows XP，因为当时候俄罗斯的高手还没有破解成熟，我每隔一段时间重装一次系统也要体验那让人兴奋的 XP 系统。那个时候很多人对 XP 都是持保留观点，甚至到现在还是，例如有一些人在 915/P4 3.06G/512DDR400/9600PRO/120G-HD 上面运行Windows 98。我觉得这种人是可悲的，而且是不可饶恕的。

软件给人的价值，在于提高工作效率，提高人与人之间的沟通效率，以及倡导新的娱乐方向。让软件使用者成为价值的创造者，同时也是新工作模式的受益者。

在这个意义来说，软件开发，是技术，某种程度上说是也一门艺术。新时期对软件工程师的要求已经不单单是会写程序，而且对其艺术修养，美学感知，音乐触觉，等等都提出了新的要求。这让我们联想到 Mac OS 这一件名符其实的“艺术品”，我想 Longhorn 正在向这个目标努力。记得以前小学语文有一篇叫做《雕凤凰》的文章，现在不知道教材改革有没有删去，我觉得那些在现阶段找 Mac 和 Longhorn 比较的人好好看看这篇文章。也许会得到一些新的观点。

 

用电脑就是用软件，我会永远记住这句话。

没有软件的电脑就像没有灵魂的人或没有生命的世界一样。好的软件赋予电脑的意义并不单单是功能的让人满足，而是一种愉快地创造价值的体验，又或者说是享受生活的绝佳体验。OS 类的软件更能体现这一点。Microsoft Windows 做到了这一点，当然我最喜欢的 Apple Mac OS 也做到了这一点。个人认为，Linux 虽然很不错，但是没有做到这一点。也许我这样说会招来很多自由软件爱好者的不屑，然而我也是热爱自由软件的，也编写着自由软件，我也把 GNU 公约看成是软件产业的唯美风景。但是我们不得不理智承认的是: Mac 和 Windows 在人性化方面走得的确已经很成熟，很成功。微软的东西通常不是最强大的，OS、Browser、Office、Design、Digital Media、Develop 等等的领域上，微软都遇到了几乎让自己崩溃的竞争对手。然而为何这样一个“帝国”依然可以屹立至今？微软软件的缺陷很多，但是这些缺陷来源于该公司对新功能，新领域，新特性的不断尝试，不断创新的初衷当中。微软的软件更新和修补的速度其实已经很快了，研发的力度也是很足够的，只是它也会遇到市场的阻力，市场消化的速度，竞争的削弱，硬件厂商的配合，还有就是人类传统科学这个强大后盾的影响。

 

软件是人类的智慧，这是我痴迷软件与编程的原因，就算是小小一个Notepad 或者 Calculator 等等现在小儿科级别的软件，却无时无刻推动着我们的社会生产；小小的 Winamp 都曾经带给过我们很多很多的欢乐，他们融入的是无差别的人类智慧，无差别的源于工程师才华对人类社会的巨大贡献。

Microsoft 是软件业界的领袖，也是 IT 产业的领袖之一，然而王者也是人，再大的公司也是由人去决策，去经营的，我们何必像那些极端分子那样，每天找它的 Bug，找他在商业运作中的不足去嘲笑，去否定，甚至去抨击呢？

 

我相信，未来的 IT 业界我们会见到多 OS 并存的状态，就像其他软件一样，是百花齐放的，是精彩纷呈的。

然而我也相信，在不久的将来，Longhorn 的发布，不会让狂热者们失望。

绝对不会。让我们拭目以待！
