随着技术的进步，市场竞争的白热化，一下子所谓的智能手机走进了越来越多人的口袋。</div>

有人说，手机是用来打电话的，偶尔发发信息，用着顺手就不错。</div>

我们先保留这种看法，等文末大家再评论也不迟。</div>

顺便另外说说一个现象，有些人一边说着“手机无用论”一边跑去买一个N7610，或者N3230用来打电话、发信息。</div>

我想现在用智能手机的主要有三种人吧。</div>

第一种：热爱IT，追求个性与功能的玩机一族；</div>

第二种：耐不住广告和各种商业活动的诱惑下不知不觉买回来的，这种人往往不知道什么是智能手机，只知道这手机比较特别，功能不错；</div>

最后一种人最可悲，为的是炫耀，他们往往也不懂得智能手机的真正特色，只是人有我有，人有我好，追求的就是用一个“天价”手机以显示“身份”。</div>

无论如何，我们去掉花俏的宣传，真正看看智能手机到底可以为我们做一点什么，精确一点说是比起传统手机，智能手机可以为我们多做一点什么。</div>
 

  • 简述智能手机</span></strong></li>
    </ul>
    笔者使用的是 Siemens SX1，基于 Symbian 60 的智能手机，同级别的还有 N6600 和 N-Gage 等。

    事实上 Symbian 是以 Nokia 为首的一系列手机厂商为了对抗 Microsoft 的 Windows Mobile，而开发的移动设备操作系统，现在已经很成熟了，第三方软件十分丰富。当然，在多媒体性能上，Windows Mobile 还是有着无可比拟的优势，但是 Symbian 以出色的性能和成熟的架构成为现金性价比最高的智能平台。当然还有其他的一些系统，但是发展程度相对落后。

    不过如何，智能手机都具备一下的几个特质：

    1. 开放的文件系统</li>
    2. 开放的软件接口</li>
    3. 多任务执行机制</li>
    4. 出色的多媒体集成</li>
      </ol>
      至于大容量的存储空间，摄像头支持，因特网浏览器等等，现在很多传统手机已经具备，不成为区别传统手机和智能手机的特质。下文中我们都会提到。

       

      • 开放的文件系统</span></strong></li>
        </ul>
        现在的主流智能手机基本上支持 MMC/SD/MS 扩容，甚至有一些手机已经用上了G级的硬盘，更大的容量和更多种类的数据要求智能手机拥有更灵活的文件管理系统。

        </a>

        基本上现在的手机浏览文件都很方便，新建文件夹，复制粘贴等等和在电脑上没有分别。

        而且搭配蓝牙、红外、USB、GPRS 等等方式，分享传输文件都变的十分方便。

         

        • 开放的软件接口</span></strong></li>
          </ul>
          无论是 Symbian 还是 Windows Mobile，现在只要懂一点C语言，都很容易在上面写软件。比起传统手机的 Java 平台，智能手机运行软件的效率很高，而且功能强大，第三方软件多姿多彩。

          我们熟悉的 Opera 也为智能手机推出特别的版本(Left)：

          </a>

          还有 NetFront 都是十分出色的手机浏览器(Right)，他们让手机不单单可以浏览普通的 WAP 站点，还让手机移动访问 www 网站一下子成为现实。手机浏览的页面，基本上和在电脑上看得网页没有多大分别，用GPRS和CDMA的手机，上网速度已经很快了。只是屏幕小，拖动实在有一点麻烦。

          除了浏览器，下面也介绍一下一些我们生活中经常要用到的几个软件，基本上都可以在智能手机上实现。

          很多人都用 Outlook 管理日程，现在地智能手机都可以方便与电脑连接，同步日程表到手机，然后在预设的时间发出提醒。

          笔者在 Outlook 03 上编辑日程表（其实更多的是课程表）和联系人资料，通过联机软件和蓝牙无线连接，很容易就同步到手机中。让我一下子变得很有时间观念：

           

          有趣的图片编辑软件(Left)，其实连著名的 Ulead 公司也为了 S60 平台开发了 Photo Express Mobile Style 软件，用于随时随地个性化编辑图片，及时有趣地和朋友分享：

          智能手机的系统开放还体现在很多细节，甚至输入法也是开放的(Right)，就像电脑上一样，你可以安装自己喜欢的输入法到你的手机，让你从此摆脱一成不变的输入发，提高文字录入速度，我想以后不会有人说那个手机“打字不快”就退而求次了。

           

          随时随地，想看就看。通过使用各种电子书软件，智能手机让我们随时随地阅读各种格式的电子书：Txt、Html、PDF 等等的格式，不需要像传统手机那样要用专门的软件把电子书转换成为 Java 可执行文件了。同时得益于分辨率相对较高，看书可以一次看好几行，比较舒服：

           

          移动办公！？通过安装 Quick Office 办公软件，智能手机可以随时随地的编辑 Word、Excel 文件，甚至使用 PowerPoint 幻灯片：

           

          有时候我也很喜欢写一些东西，或者在车上，或者在路上，不方便开电脑，也不喜欢用纸笔。打开手机的 Yedit 软件，和电脑上的“记事本”一样，快速记录你瞬间的灵感。

           

          • 多任务执行机制</span></strong></li>
            </ul>
            智能手机可以同时执行多个任务，一边听音乐，一遍打开办公软件，还可以一遍收发短消息，一边打电话。至于可以同时执行多少程序，就看你的手机机能了。

            不再需要为了某一功能而退出当前的功能了。能够在手机上一边发信息，一边听MP3...我想这是智能手机一大亮点。

             

            • 出色的多媒体集成</span></strong></li>
              </ul>
              强大的机能让我们的移动游戏更好玩：

               

              雷曼(Left)，3D 赛车(Right)，还有如超音鼠，古墓丽影，甚至 FIFA 等等的明星游戏都已经进驻智能手机市场。3D 游戏，在线游戏这些已经不是 PC 游戏和专业游戏机的专利了，手机同样可以 Online，可以 3D！传统手机的 Java 游戏不但执行很慢，而且视觉音效也不能让人满意。在智能手机的游戏世界，我们还可以通过蓝牙进行无线即时对战。喜欢玩 GB 和 FC 游戏朋友也会觉得智能手机很神奇，因为已经有很多针对智能手机开发的模拟器，可以满足你在手机上玩 GB 和 FC，甚至SFC游戏的梦想了。如果你是游戏狂热分子，我想 Nokia 的 N-gage 系列智能手机会是你理想的伙伴。

               

              出色的媒体集成还包括 MP3、FM Radio 的集成：

              笔者的 SX1 支持立体声 MP3 播放，也是我每天必用的功能之一，音质还算满意。我喜欢音乐，智能手机可以给我随时随地被音乐包围的感觉。

               

              通过集成 RealOne Player 在旅行中我们可以在车上欣赏电影。看起来很不实在，但是现在的电话真的已经做到了：

               

              • 最后</strong></span></li>
                </ul>
                看了这么多，或者有一些人觉得这些还很遥远，但是告诉大家，这些已经发生在我们身边。不再活在科幻小说中，技术的进步在我们还没有准备好的时候已经一下子飞到了这个时代。手机已经是一种生活工具，包含的也不仅仅限于发信息，打电话了。

                再过不多久国内 3G 网络就开始运营了，移动视频通话也就有了网络支持。随着 GPS 定位系统的民用化，越来越多的手机厂商开始集成。手机的世界是一天比一天精彩了。客观说现在的智能手机的确存在种种的不足，例如电池的持航能力不足，售价难以平民化，性能仍然存在瓶颈等等。但是“智能”的未来是可以遇见的。

                然而，也不是说强大就是好，我对《手机》中的情节记忆犹新，真正的沟通其实并不依赖任何工具，手机已经不能成为我们的沟通工具了（沟通和通讯是不同的），它是一种新的生活工具，从固有的概念中独立出来，成为一种崭新的生活方式的中心。如果要说沟通，主导沟通的是我们的内心，而不是手机。

                崭新的通讯时代已经到来了，我们准备好了吗？
