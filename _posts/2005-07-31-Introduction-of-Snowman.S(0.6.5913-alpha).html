Introduction of Snowman.S

Snowman.S,  Enjoy Smart Media
体验智能多媒体

Version：0.6.5913 alpha

首先欢迎您使用 Snowman.S，对您的支持表示感谢。

-----------------
What's Snowman.S?
-----------------

Snowman.S 源自 Snowman Media 媒体播放软件，Snowman Media 在 5.0 版本后改名为 Snowman。
“.S” 是 Snowman 6.x 的一个新的系列，灵感来自 iPod shuffle，强调的是音乐的易用性和体验性。

----------------------------------------------
我们有时候打开播放器甚至会花时间想：今天听什么歌？
----------------------------------------------

在这个电脑和因特网已经泛滥的时代，我们基本上离不开电脑。
自然，电脑娱乐的概念在这几年显得格外红火。

就一线媒体软件 Windows Media Player，RealPlayer，QuickTime 来说，
这三个播放器的优势在于拥有自己开发的媒体格式，基本上拥有固定的使用群。
而且由于很多第三方播放器需要调用其解码器以解码多媒体资料，
故这三个播放器的普及率十分高。
由于商业化的支持，他们的发展得到多方面的配合，功能和拓展性都非常好。
兼用播放器方面也还有很多优秀的作品如 Winamp 和国内的 千千静听 等等，
还有可以灵活调用解码器和分离器，简单但功能强大的 Media Player Classic。
在这样一个竞争环境中，对于做一个兼用功能的播放器，似乎很难超越这些前辈们。
然而发觉，很多做得强大的播放器会越来越像 WMP，Rral 它们那种模式。
领跑的已经不单单是技术了，而是一种娱乐文化。

于是我开始重新思考播放软件，发现除了一些特殊的环境，在我们的生活中，
我们很少用到主流播放器的所有功能，而且在现在大容量硬盘普及，到处 P2P 的年代，
上 G 的音乐收藏在很多电脑中是随处可见了。
我们有时候打开播放器甚至会花时间想：今天听什么歌？
Snowman.S 在开始构思的时候就针对用户的这些疑惑，始终以用户体验为设计重点。
通过尽量减少用户的操作，力求照顾不同用户的个性特点，
融合了很多心理学和听觉心理学的成果。

我试图做出一个可以让用户觉得体贴的播放器。
当然现在 Snowman.S 还处于 alpha 测试的阶段，很多细节需要慢慢完善。
从几年前的 Windows XP 发布到现在，很多软件都把版本号取 XP，
而我，却看不到多少真正懂得 XP 内涵的软件作品。
强大不是一切，功能不代表效率。
体贴用户才是根本，希望我可以给你们带来另类的，愉悦的软件体验，
如果这个版本让您失望了，希望您可以反馈您的真实感受，
我会把下一个版本做得更好。

-----------------------------------------------------------------------------------------------
Snowman.S 会找到您喜欢的音乐，并增加重放次数；Snowman.S 会知道你讨厌那些音乐,减少该音乐出现的频率...
-----------------------------------------------------------------------------------------------

Snowman.S 无论功能和使用上与普通的播放软件有很大的差别。
表面上看 Snowman.S 是一个极其简化的音频播放软件。
内核上，Snowman.S 拥有比市面上所有播放软件都要高的 AI。

Snowman.S 后台应用很多的人工智能的思想智能地管理播放程序。
AI 程序将分析用户的行为并根据行为特征分析结果设定音乐的播放顺序，播放次数以及播放时间，音量等等。
Snowman.S 对用户的分析了解将随着时间的推移完善。
您会发现 Snowman.S 会越来越了解您。

然而这一切都是自动的，整个过程不需要用户作调整，您只需要告诉它您的音乐放在什么地方，然后只管欣赏就可以了。
Snowman.S 会找到您喜欢的音乐，并增加重放次数；Snowman.S 会知道你讨厌那些音乐,减少该音乐出现的频率...
套一个 Apple 的说法：“你会觉得音乐从此变得主动！”
还有很多细节会迎合您的播放习惯，慢慢会越来越适应您的要求。

Why not build it on .Net Frameworks?
------------------------------------

Snowman.S 最初计划在 .Net Frameworks 2.0 上开发，
程序写了一大半但是由于目前 .Net 2.0 的开发工具 VS2005 不很稳定，
而用 .Net 1.1 做又觉得很费劲，对用户也是一种负担，所以还是选择 VS6。
其实本人很喜欢 .Net 2.0 用它开发效率很高。
下一个版本的 Snowman 应该会是在 .VS2005 上完成的。

Something else
--------------

03年 Snowman Media 4 ilxz 在太平洋和华军发布后，
得到了很多人的支持，好几个网站都转载了。
很多用户都联系了我，和我谈了很多，关于 Snowman 的优缺点，还有问我以后的打算。
他们都希望我可以继续开发新的版本，但是由于种种原因，一直没有动手。
到大学后，开始做 Snowman Media 5.0 取名 Snowman.F(flora)，
Snowman.F 是一个完全针对网络的播放软件，在 .Net 架构上开发。
已经基本上接近完工，并小规模开始发放测试。

一年半之前由于申请休学一年外出尝试创业，Snowman.F 夭折。
这一年是我生命的转折点，也曾经创下很多看似疯狂的举动，但更多的是我生命中的寒冬。
这一年中，我失去了很多，甚至一段我至今无法释怀的爱情，和曾经推心置腹的友谊。
梦想破灭，一个个希望被粉碎，一份份信任被扭曲...
但我想我得到的会是更多，那种心灵的成长，说不清楚，却会让我一辈子受益。

在这一年中，特别是在深圳的大半年，我接触到很多关于心理学，关于设计的理念，
也了解到催眠，知道人的情绪和音乐密切相关，也知道不同的人格对音乐有不同的要求。

半年前创业失败回到华师大继续学业，从国际会计转到电子商务专业。
同期接触到 iPod shuffle 对多媒体的观念，被小小的 iPod 彻底改变。
于是重新构思新的媒体软件，构思了很多方案，最终确定 Snowman.S。
原本信心不大，但前段时间由于要完成一份设计和一个在 U.K. 留学的的朋友 xiaoxiao 谈的时候，
他给我一句话：“知道飞利浦最新的广告语是什么吗？科技就像打开盒子一样简单。”
于是我把一直以来构思写成程序，完成了这个这个版本的 Snowman.S。
虽然将来还要花很多时间去完善，因为人很难把握，做到真的体贴用户其实真的很难。
我只能慢慢努力。
做软件是很辛苦的，不像要赶一份设计，几个通宵然后交给别人就OK了，
维护一个软件会花很多的精力，而且我没有一个团队，所以也不知道可以走多远。
尽力吧，不知道是不是我喜欢听音乐，喜欢看电影，在我刚开始接触电脑到现在，
都很狂热播放软件，因为觉得在播放软件面前，大家都是快乐的。
这也是我一直在写播放软件的动力。
最后希望 Snowman 可以让大家快乐！
谢谢您的支持！
